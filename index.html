Про нас

Ми виробляємо крейду та глину, які подобаються дітям, вчителям і майстрам. Збалансована твердість, чисті пігменти, зручний формат.

  • Нетоксичні склади та контроль якості
  • Підходить для школи, дому та студій
  • Оптові замовлення для навчальних закладів

Галерея

Питання та відповіді

Чи безпечна крейда для дітей?

Так, формула нетоксична. Рекомендуємо користуватися під наглядом дорослих та уникати вдихання пилу.

Залишає пил — що робити?

Ми використовуємо склад із зменшеним пилом. Для дошок — протирання вологою ганчіркою після занять.

Чи підходить глина для випалу?

Натуральна глина підходить для ліплення. Для випалу уточніть технічні параметри перед замовленням.
